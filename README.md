# Hunter Barbershop Platform

Платформа онлайн-бронирования и CRM-система для барбершопа «Хантер».

## О проекте

Веб-сайт для онлайн-бронирования стрижек с внутренней CRM-системой для мастеров. Позиционирование — дерзкая мужская философия «На охоту с охотой».

## Сайт

**URL:** https://slava-hunter.ru/

**Админ-панель:** https://slava-hunter.ru/admin

**Хостинг:** Timeweb

## Технологический стек

| Компонент | Технология |
|----------|-------------|
| Frontend | React + Vite + TypeScript + Tailwind CSS |
| Backend | Next.js 14 (App Router) + Prisma |
| База данных | PostgreSQL |
| Менеджер пакетов | Bun |

## Структура проекта

```
hunter-club/
├── frontend/          # Клиентская часть (публичный сайт)
│   ├── src/
│   │   ├── components/   # UI компоненты
│   │   ├── pages/        # Страницы
│   │   ├── context/      # React контексты
│   │   ├── hooks/        # Кастомные хуки
│   │   └── lib/          # Утилиты

│
├── backend/           # Серверная часть (CRM)
│   ├── src/
│   │   ├── app/          # Next.js страницы и API роуты
│   │   │   ├── admin/    # Админ-панель
│   │   │   ├── api/      # API эндпоинты
│   │   │   └── page.tsx  # Публичные страницы
│   │   ├── lib/          # Бизнес-логика
│   │   ├── prisma/       # Схема базы данных
│   │   └── types/        # TypeScript типы
```

## Функциональность

### Публичный сайт (Frontend)
- Hero-секция с главным оффером
- Блок философии и позиционирования
- Команда мастеров с грейдами и ценами
- Модуль онлайн-бронирования
- Многоязычная поддержка

### Админ-панель (Backend)
- Управление бронированиями
- Управление клиентами
- Управление услугами
- Расписание и график работы
- Отзывы и их модерация
- Аналитика
