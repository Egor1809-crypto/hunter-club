## 4. Требующиеся доработки

### 高 Приоритет

1. **Создать пользователя-барбера в CRM**
   - После деплоя выполнить SQL-запрос в БД
   - Создать пользователя с логином и паролем

2. **Настроить SSL (HTTPS)**
   - Установить certbot на сервере
   - Получить сертификат для slava-hunter.ru

3. **Настроить nginx для production**
   - Создать конфиг в репозитории
   - Настроить проксирование

### 中 Средний приоритет

4. **Добавить соцсети в футер**
   - Telegram
   - Instagram
   - (ссылки ожидаются от клиента)

5. **Создать nginx конфиг**
   - Фронтенд (статика)
   - Бэкенд (API)
   - SSL terminating

6. **Обновить deploy.sh**
   - Добавить nginx конфиг
   - Проверить пути

### 低 Низкий приоритет

7. **Sitemap.xml для SEO**
8. **Проверить robots.txt**
9. **Обновить .env.example**
